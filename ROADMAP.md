# FluxArr Roadmap

## Planned Features

### Request & Approval Workflow
Allow users to request TV shows and movies, with admin review and approval.

- Users can submit requests for content not yet in Sonarr
- Requests queue visible to admins
- Admins can approve, reject, or mark requests as pending
- Notifications when requests are processed
- Request history and status tracking

### Enhanced Show/Movie Details
Display comprehensive cast and crew information on detail pages.

- Actors with character names and photos
- Directors and producers
- Writers and creators
- Guest stars per episode
- Links to actor filmographies

### Extended Filter Options
Add filters based on the new cast/crew data.

- Filter by actor
- Filter by director
- Filter by producer/creator
- Combine with existing filters (genre, rating, etc.)

### Searchable Filters
Make filter dropdowns searchable for easier selection.

- Type-ahead search in genre dropdown
- Type-ahead search in network dropdown
- Type-ahead search in language dropdown
- Type-ahead search in cast/crew filters

### Show Descriptions on Discover Page
Display brief summaries directly on show cards in the grid view.

- Truncated description preview on hover or always visible
- Configurable in user settings (show/hide)
- Mobile-friendly display
